# 理解this

this是最终指向的是调用它的对象

```
function a(){
  var user = "dreamer";
  console.log(this.user);           //undefined;
  console.log(this);                //Window
}

a();

var o = {
  user:"dreamer",
  fn:function(){
    console.log(this.user);         //dreamer
  }
}

o.fn();
```

this指向的是对象o，this的指向在函数创建的时候决定不了，在调用的时候才能决定

在Javascript里面，this指针代表的是执行当前代码的对象的所有者。
